C++: Functions and References We carry on with the core C++ syntax, covering the key topics of functions and references.
Build Automation with CMake In the second part of the day, we start to modularize the functionality in our program to make it easier to work with and give it logical structure. We then use the CMake build tool to automate the compilation of the program from the component sources, break the build down into a main application and an implementation library, and finally package everything up.
C++: File I/O and the Caesar Cipher In the third part of the day we examine how to read from and write to files in C++ and finally we implement the Caesar cipher
